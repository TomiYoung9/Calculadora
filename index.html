🧠 Algoritmo (ASE 2025)

1) Parámetros pilares (RS — algoritmo universal, independiente de FEVI)

e′ (ajustado por edad), E/e′ (promedio ≥14, septal ≥15 o lateral ≥13) y TR ≥2.8 m/s o PSAP ≥35 mmHg.

2) Decisión y gradación (RS)

  • 3/3 alterados → PAI elevada → Grado II si E/A <2; Grado III si E/A ≥2.
  • 0/3 alterados → PAI normal. Si además e′ reducido + E/A ≤0.8 → Grado I.
  • 1–2/3 alterados → evaluar parámetros adicionales:
    • LARS ≤18%, PV S/D ≤0.67, LAVi >34 ml/m² (Área >20 cm²) o IVRT ≤70 ms
    • Si alguno cruza umbral → PAI elevada (Grado II si E/A <2; Grado III si E/A ≥2)
    • Si ninguno cruza → PAI normal. Si además e′ reducido + E/A ≤0.8 → Grado I
    • Si no se ingresó ningún adicional → indeterminado

Con FEVI <50%: la disfunción diastólica se asume. El grado se estima con los mismos criterios (algoritmo universal ASE 2025, Fig. 3).

3) FA (criterios pilares)

  • E ≥100 cm/s; septal E/e′ >11; TR ≥2.8 m/s o PSAP ≥35; DT ≤160 ms

≥3 → PAI elevada; 0–1 → PAI normal; 2 → apoyo con LARS ≤18% y/o PV S/D <1 (si ambos → elevadas; si 0–1 → indeterminado).

4) MAC

E/A <0.8 → PAI normal; E/A >1.8 → PAI elevada; si 0.8–1.8 → usar IVRT (≥80 ms normal, <80 ms elevadas).
